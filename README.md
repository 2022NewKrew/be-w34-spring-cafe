- [X] 로그인한 사용자만 게시글의 세부내용을 볼 수 있다.

- [ ] 로그인 사용자와 게시글 작성자가 같은 경우 삭제 가능하다.
- [ ] 댓글이 없는 경우 삭제가 가능하다.
- [ ] 게시글 작성자와 댓글 작성자가 다를 경우 삭제는 불가능하다.
- [ ] (선택) 게시글 작성자와 댓글 작성자가 모두 같은 경우 삭제가 가능하다.
- [ ] (선택) 게시글을 삭제할 때 댓글 또한 삭제해야 하며, 댓글의 삭제 또한 삭제 상태(deleted)를 변경한다.
