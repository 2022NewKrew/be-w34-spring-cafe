### 📌 URL Convention

| url               | 기능                  |
|-------------------|---------------------|
| `GET` /user       | List all users      |
| `POST` /user      | Create a new user   |
| `GET` /user/form  | Get a register form |
| `GET` /user/:userId | Get a user profile  |
